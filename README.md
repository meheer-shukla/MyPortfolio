# 🚀 Meheer Shukla — Portfolio Website

A futuristic, high-end developer portfolio built with pure HTML, CSS, and JavaScript.

Inspired by cyberpunk aesthetics and modern minimalism, this portfolio combines smooth UI interactions, glassmorphism, ambient lighting effects, and responsive design into a premium developer experience.

---

## ✨ Features

- 🌑 Dark / Light Theme Toggle
- 🎯 Custom Animated Cursor
- 📱 Fully Responsive Design
- 🧊 Glassmorphism UI Components
- ⚡ Smooth Navigation & Interactions
- 🍔 Fullscreen Mobile Navigation Menu
- 💼 Projects Showcase Section
- 🧠 Skills & Tech Stack Display
- 📬 Functional Contact Form Integration
- 🎨 Modern Typography & Visual Hierarchy
